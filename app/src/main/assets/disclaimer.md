# 古籍阅读 - 使用说明

## 简介
古籍阅读是一款专注于中国古典文学的阅读应用，内置四大名著、唐诗宋词、历史典籍等资源。

## 主要功能
1. **书架管理**：添加、删除、整理书籍
2. **在线搜索**：通过内置书源搜索书籍
3. **本地导入**：支持TXT、EPUB等格式的本地书籍
4. **书源管理**：导入、导出、编辑书源

## 内置书源
- 中国哲学书电子化计划（ctext.org）
- 古诗文网（gushiwen.cn）
- 维基文库（wikisource.org）

## 如何添加书源
1. 打开应用，进入「书源管理」
2. 点击右上角菜单，选择「导入书源」
3. 选择JSON格式的书源文件
4. 确认导入

## 如何搜索书籍
1. 点击底部导航栏的「发现」
2. 选择书源分类
3. 输入关键词搜索
4. 点击书籍添加到书架

## 本地书籍导入
1. 将TXT、EPUB等格式的书籍文件放入手机存储
2. 打开应用，点击「导入」按钮
3. 选择文件即可

## 书源格式说明
书源文件为JSON格式，包含以下主要字段：
- bookSourceName：书源名称
- bookSourceUrl：书源网址
- searchUrl：搜索地址
- ruleBookInfo：书籍信息规则
- ruleToc：目录规则
- ruleContent：正文规则

## 注意事项
1. 部分书源可能需要网络连接
2. 书源有效性可能随时间变化
3. 建议定期更新书源
4. 请尊重版权，支持正版
